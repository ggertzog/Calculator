## Начало работы
### Склонируйте репозиторий к себе на устройство с помощью команды в терминале
* `git clone (адрес репозитория)`;
### Подтяните зависимости для корректной работы приложения
* `npm ci или npm i`
### Для сборки проекта в папке public 
* `npm run build`
### Для запуска проекта 
* `npm run start`
